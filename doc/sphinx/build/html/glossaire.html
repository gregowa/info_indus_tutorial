Glossaire

Logiciels

ROS 2

Robot Operating System 2 (ROS 2) est un ensemble de bibliothèques et d’outils qui aident les développeurs à créer des applications robotiques.

DynamixelSDK

Dynamixel Software Development Kit (SDK) est une bibliothèque qui permet de contrôler les moteurs Dynamixel.

Sphinx

Sphinx est un générateur de documentation qui convertit des fichiers reStructuredText en divers formats de sortie, y compris HTML et PDF.

GitHub

GitHub est une plateforme de développement collaboratif pour héberger et réviser du code, gérer des projets et construire des logiciels aux côtés de millions d’autres développeurs.

Python

Python est un langage de programmation interprété, interactif et orienté objet, connu pour sa lisibilité et sa simplicité.

Vocabulaire

Virtual Environment

Un environnement virtuel est un outil pour garder les dépendances requises par différents projets dans des espaces séparés, en évitant les conflits de versions et facilitant la gestion des paquets.

SSH

Secure Shell (SSH) est un protocole réseau cryptographique pour l’exploitation de services réseau de manière sécurisée sur un réseau non sécurisé.

reStructuredText (reST)

reStructuredText (reST) est un langage de balisage léger utilisé principalement dans la documentation technique. Il est conçu pour être facile à lire et à écrire, et est souvent utilisé avec Sphinx pour générer de la documentation en divers formats.
