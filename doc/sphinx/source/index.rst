################################################
Projet d'Informatique Industrielle avec ROS 2
################################################

Le projet de ce semestre consiste à mettre en application ce que nous avons appris sur Github, ROS 2 et les documentations Sphinx. Pour celà, nous allons réaliser un tutoriel sur l'utilisation de la librairie DynamixelSDK.

=================
Ressources utiles
=================

- `Sphinx <https://www.sphinx-doc.org/en/master/>`_ :
  Documentation officielle de Sphinx, un générateur de documentation.

- `Cours d'Informatique Industrielle avec ROS2 <https://yguel.github.io/informatique_industrielle_avec_ROS2/index.html>`_ :
  Cours complet sur l'informatique industrielle avec ROS 2.

- `ROS 2 Humble Documentation <https://docs.ros.org/en/humble/>`_ :
  Documentation officielle de ROS 2 Humble.

- `Dynamixel Quick Guide for ROS 2 <https://www.youtube.com/watch?v=E8XPqDjof4U>`_ :
  Guide rapide pour utiliser Dynamixel avec ROS 2.
